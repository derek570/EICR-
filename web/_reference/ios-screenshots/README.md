# iOS reference screenshots

Drop or paste iOS screenshots here for visual parity checks.

Use kebab-case, match the web route where possible:

- `login.png`
- `dashboard.png`
- `job-detail-installation.png`
- `job-detail-circuits.png`
- `job-detail-observations.png`

## Resolution

iPhone 14 / 15 Pro (1179×2556 @ 3x) is the primary target. Anything from iPhone 12 and up is fine — I'll scale internally.

## Workflow

1. Screenshot in the iOS app.
2. Paste into the conversation or drop the file in this folder.
3. I'll write it here with the correct filename if pasted in chat.
4. Before starting each UI phase, I'll enumerate the shots present in this directory and compare them to Playwright captures of the web build.
