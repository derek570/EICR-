# Mac Shortcut Scripts

Double-click .command files on macOS to run them.

- `Create User.command` — Create a new user account
- `Manage Users.command` — Manage existing user accounts
- `Open Editor (Derek).command` — Open the certificate editor for Derek
- `Open Editor (Michael).command` — Open the certificate editor for Michael
- `Setup Mac (Michael).command` — Initial Mac setup for Michael
